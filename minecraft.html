Minecraft Server Information

Welcome to my new server! This server is accessible by both Java and Bedrock editions, using Java mechanics.

How to Join for Java

  • Open Minecraft and go to Multiplayer.
  • Click on Add Server.
  • Enter 104.238.205.83:28018 as the server address.
  • Click Done and join the server.

How to Join for Bedrock

  • Ensure you're on a device that supports custom servers (mobile, PC, etc.).
  • Go to Play > Servers.
  • Scroll down and click on Add Server.
  • Enter 104.238.205.83 for the Server Address and 28018 for the Port.
  • Click Save and join the server.
